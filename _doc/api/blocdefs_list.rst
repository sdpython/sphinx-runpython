===================================
blocreflist, exreflist, mathdeflist
===================================

These extensions can recall the blocks defined by
:ref:`l-blocdefs`.

exreflist
=========

::

    .. exreflist::
        :tag: example
        :contents:

Which gives:

.. exreflist::
    :tag: example
    :contents:
