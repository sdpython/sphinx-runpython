===================================
blocreflist, exreflist, mathdeflist
===================================

These extensions can recall the blocs defined by
:ref:`l-blocdefs`.

exreflist
=========

Example:

::

    .. exreflist::
        :tag: example
        :contents:

Which gives:

.. exreflist::
    :tag: example
    :contents:

blocreflist
===========

An example:

::

    .. blocreflist::
        :tag: bloc
        :contents:

Which gives:

.. blocreflist::
    :tag: bloc
    :contents:

mathdeflist
===========

An example:

::

    .. mathdeflist::
        :tag: definition
        :contents:

Which gives:

.. mathdeflist::
    :tag: definition
    :contents:
